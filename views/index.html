About

Scryfall Stream Extension provides a "Stream" button on Scryfall search result pages to allow you to change an on-stream card graphic with no special equipment or setup.

Alternate uses can include sharing card imagery while playing Magic via teleconference during a hypothetical pandemic. Just have both players open up the card display page.

Instructions

The extension uses the concept of "rooms" to allow multiple people/browsers to control the same image output. Set a room identifier, either uniquely or using a shared room.

Simply add a Browser Source to your streaming client of choice with the url provided by the extension.

The webpage will automatically size to fit the size of your source, so try to match a standard card resolution such as 745x1040 (Width: 745px, Height: 1040px). The extension uses the highest resolution offering Scryfall has, so it should scale well for common stream resolutions. Do note that some Scryfall images differ in size, but any stretching should be minor. Full resolution images are not available for some foreign languages.

Development

This extension is backed by a cheaply hosted Node server using in memory storage because I'm too cheap to pay for a store. If it crashes or I push a deployment it will wipe out all active imagery, should this develop into something more then I'll reapproach that limitation.

Source is available on Github, just don't read too closely.
